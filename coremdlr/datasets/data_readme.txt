DATA README

1. Well Log Data (.las files)
Well Log Data is provided from UK Oil and Gas Association (https://www.ogauthority.co.uk/)

From the National Data Repository (https://ndr.ogauthority.co.uk/)

2. Core Images 

The images are altered from the British Geological Survey (https://www.bgs.ac.uk/)

More specially, from this search (https://www2.bgs.ac.uk/data/offshoreWells/wells.cfc?method=searchWells)

3. Color Channel Logs

These are caluclated from the images. These are a derivative of an altered data type.
